#Tarefa_Controle_De_Fluxo

- Este Tarefa simula um programa que faz a Leitura no console de 4 notas e divide por 4 além disso o programa verifica se a nota é maior ou igual a 7 (imprime na tela Aprovado), se é maior ou igual a 5 e menor que 7 (imprime Recuperação), se caso a nota não seja maior ou igual a 5 (imprime Reprovado).
